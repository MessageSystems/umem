umem by MessageSystems

This is a port of the Solaris umem memory allocator to other popular operating systems, such as Linux, Windows and BSDish systems (including Darwin/OSX).

libumem is a slab allocator with a number of debugging options taken from OpenSolaris and made to run on a variety of other systems. You can read more about using umem here: http://developers.sun.com/solaris/articles/libumem_library.html And more about Jeff Bonwicks earlier work on the kernel equivalent here: http://www.usenix.org/publications/library/proceedings/bos94/bonwick.html

Dependencies

Builds on Linux, Mac OS/X, FreeBSD. Contains code that runs on Windows, but currently does not ship with build/project files for Windows--contributions welcomed!

Install

Run ./autogen.sh to generate the configure script. Then run ./configure ; make ; make install
